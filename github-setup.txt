# GitHub Setup Commands
# After creating your GitHub repository, run these commands:

# 1. Add the remote repository (replace Pallav46 with your GitHub username)
git remote add origin https://github.com/Pallav46/pallav-portfolio.git

# 2. Push your code to GitHub
git push -u origin main

# 3. Your repository will be available at:
# https://github.com/Pallav46/pallav-portfolio
